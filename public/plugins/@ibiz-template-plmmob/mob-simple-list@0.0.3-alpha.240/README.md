# 移动端简单列表

该插件基于列表部件增强，主要用于显示数据选择的时候集中显示已选择数据所用，正常情况下该列表不在界面上显示。**该插件隶属于列表绘制插件（基于列表部件进行扩展）**

## 功能说明

- 该插件主要是用来临时保存数据选择视图已选择的数据，当关闭视图时，数据选择面板就会从简单列表上获取已选择的数据并抛出去，供外部使用。
